what it does

что делает

send link from tiktok, instagram, twitter or youtube. get video back without watermarks.

присылаешь ссылку из tiktok, instagram, twitter или youtube. получаешь видео без водяных знаков.

how i made it

used python and aiogram library. for downloading used yt-dlp.

использовал python и библиотеку aiogram. для скачивания использовал yt-dlp.

hardest part: websites block ip addresses. had to use proxies and retry logic.

сложнее всего: сайты блокируют ip адреса. пришлось использовать прокси и логику повторных попыток.
