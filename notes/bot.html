download bot

бот для скачивания

2025-02-10

built a bot for downloading videos from social networks. simple, fast, no watermarks.

написал бота для скачивания видео с соцсетей. простой, быстрый, без водяных знаков.

features

фичи

- supports: tiktok, instagram, twitter/x, youtube
- no registration required
- instant file delivery
- runs 24/7

- поддержка: tiktok, instagram, twitter/x, youtube
- без регистрации
- мгновенная отправка файлов
- работает 24/7

stack

стек

python 3.11
aiogram 3.x
yt-dlp
redis

main issue — ip blocking from platforms. solved via proxy rotation and graceful retry with exponential backoff.

главная проблема — блокировка ip со стороны платформ. решил через ротацию прокси и graceful retry с экспоненциальной задержкой.

result

результат

~500 downloads per day. not open source yet — using myself and sharing with friends. might publish on github someday.

~500 загрузок в день. пока не открытый исходник — использую сам и даю друзьям. возможно, когда-нибудь выложу на github.
