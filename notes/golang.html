golang thoughts

мысли о golang

2025-01-28

two months trying to learn go. some things are good, some are weird.

два месяца пытаюсь учить go. кое-что хорошее, кое-что странное.

good

хорошее

  • one file to deploy, no dependencies hell
  • один файл для деплоя, нет ад зависимостей
  • easy to make things work at the same time
  • легко делать чтобы вещи работали одновременно
  • standard library has almost everything
  • стандартная библиотека почти всё содержит
  • one style for everyone, no arguing
  • один стиль для всех, без споров

weird

странное

  • checking errors every line is annoying
  • проверять ошибки каждую строчку раздражает
  • no normal way to make generic functions
  • нет нормального способа делать универсальные функции
  • modules are confusing
  • модули запутанные
  • too many nil checks
  • слишком много проверок на nil

will i continue?

буду ли продолжать?

yes. for backend it's good. for other things i'll use python.

да. для бэкенда хорош. для другого буду использовать python.
